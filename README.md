# ☠️ Causes of Death Worldwide — What's Actually Killing Us?

**Author:** Fatima Zahra Boutkhil
**Tools:** Python (Pandas, Matplotlib, Seaborn) + SQL (SQLite)
**Data:** Our World in Data

## About
An analysis of global causes of death across countries and decades 
to uncover surprising patterns — what we've solved, what's getting 
worse, and how Morocco compares to the world.

## Questions Explored
1. What are the top 5 causes of death globally?
2. How have causes of death changed over time?
3. Which cause increased the most in the last 20 years?
4. Which cause decreased the most — what did we solve?
5. What is the leading killer by region?
6. How does Morocco compare to the global average?

## Key Findings
- Cardiovascular disease kills almost double the next cause (Cancer)
- We're not getting less safe — we're getting older
- HIV/AIDS deaths dropped 45% since 2000
- Every region in the world shares the same #1 killer
- Morocco's cardiovascular death rate is above the global average
